Torsional Irregularity Check as per IS 1893 (Part 1): 2016

Problem Statement

Perform a torsional irregularity check for a building with the following details:

  • Plan Dimensions: 30m (X) × 40m (Y)
  • Height: 45m (15 stories, 3m/story)
  • Seismic Zone: Zone III
  • Structural System: RC Frame with Shear Walls

Assumptions

Parameter Value
Column Size 600mm × 600mm, 6m spacing
Shear Walls Wall 1: (6m, 0-6m), Wall 2: (24m, 34-40m), 6m long, 300mm thick
Seismic Weight/Floor 12,000 kN
Soil Type Type II (Medium)
Importance Factor (I) 1.0
Response Reduction Factor (R) 5

Calculations

Step 1: Centers of Mass and Rigidity

CM: (15m, 20m), CR: (15m, 25m), e_sy = -5m (Y-direction).

Step 2: Seismic Base Shear

Z = 0.16, T_y = 0.64s, S_a/g = 2.5, A_h = 0.04, V_B = 7,200 kN.

Step 3: Accidental Eccentricity

Y-direction: e_dy = -6m (1.5 × -5 + 1.5).

Step 4: Displacement Analysis (Top Floor)

  • Force: 960 kN (Y-direction)
  • Torsional Moment: 5,760 kNm
  • Rotation (θ): 0.00524 rad
  • Displacements:
    • X = 0m: 40.2cm
    • X = 30m: 24.6cm

Step 5: Torsional Irregularity Check

Δ_max = 40.2cm, Δ_avg = 32.4cm, Ratio = 1.24.

Result

Δ_max / Δ_avg = 1.24 > 1.2 → Torsional Irregularity Exists.

Recommendation: Perform dynamic analysis per Clause 7.7.
